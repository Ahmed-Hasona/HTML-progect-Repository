Database is a collection of interrelated data that allows to effectively gather, insert and remove data from databases and organizes the data in tables, views, schemes, reports

For example, university database organizes undergraduate, faculty, and admin personnel data. and helps to easily access, upload, and erase data..

A database consists of:

Database structure A unified object composed of a set of interrelated data is constructed as. Repository Includes rules for users to access data and rules for organizing data , i.e. metadata.

End users Who access the database.

Interfaces Which are the different ways in which an end-user can access the data through display terminals, remote terminal phone links, Touch screens, barcode readers, using voice commands, touch pads, graphics selection, keyboard or mouse menu selection, Command keying or card scanning.

Database Management System: Database Management System ( DBMS) is the software which is used to handle database. For eg, MySQL, Oracle is a common commercial DBMS which is used in various applications.

DBMS allows consumers to execute the following functions:

1-Data Definition: It helps to create, modify and delete definitions that define data organization in database.

2-Data Update: It allows to upload, change and uninstall the real data in the database.

Computer Assisted Information Development Tools (CASE tools) A component of the database system is also considered. Those are software tools that assist with program architecture and database construction.

Database Administrators (DBA) Manage the DBMS and are responsible for an organisation's internal database infrastructure. In a later section you'll look more thoroughly at the DBA 's position.
